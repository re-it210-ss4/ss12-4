Phần 1: Phân tích
Spring Boot quản lý cấu hình mặc định thông qua cơ chế Auto-configuration. Các giá trị mặc định được định nghĩa trong file spring-configuration-metadata.json bên trong các file JAR của Spring.

Bạn có thể tìm thấy danh sách đầy đủ tại trang chủ Spring Common Application Properties.
